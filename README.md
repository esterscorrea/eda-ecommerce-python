# 📊 Análise Exploratória de Dados — E-commerce

Projeto de EDA completo sobre vendas de uma loja online em 2018 com mais de 51.000 registros.

## 🔍 Etapas da Análise
- Visão geral dos dados
- Estatísticas descritivas
- Limpeza e tratamento de valores ausentes
- Análise e visualização de dados
- Matriz de correlação

## 📈 Principais Insights
- Fashion é a categoria mais vendida (~$11M em vendas totais)
- Cartão de crédito é o método de pagamento dominante (74% das transações)
- Vendas cresceram significativamente a partir de março/2018
- Forte correlação entre preço de venda, lucro e custo de envio (0.92)
- Distribuição de vendas é bimodal com picos em $50-90 e $210-230
